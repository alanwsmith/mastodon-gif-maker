Mastodon GIF Maker

Hi there!

I was disappointed to learn Twitter has started prohibiting links to other social networks. I want to let me followers know I've moved to Mastodon, but I don't want to run afoul of their guidelines. So, I setup this GIF maker that will let me send the message. You're welcome to use it too.

Enter your message then click "Create it" to do just that. (And make sure to read the notes, this is brand new and still a little clunky.)






(Note: the image is generated in a pop-up window which your browser will probably block the first time. Give it permission and try again and it should work)

Save the image from the pop-up window and then you can post it on twitter. It'll look link this. (NOTE: you'll probably need to add the filename including `.gif` when you save the file. I'm working to fix that.)


This is a super basic first version. I'm still adding features which you can see further below.
